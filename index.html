Extract ZIP Files

Once downloaded, extract the ZIP archives on your computer to access all template files.

  • Right click the ZIP file on your computer
  • Click "Extract Here" or "Extract All"
  • Open the extracted folder
  • Locate the JSON template files inside

03

Open WordPress Dashboard

Login to your WordPress admin panel and navigate to the Elementor Templates section.

  • Login to WordPress Admin Panel
  • Go to Templates in the left sidebar
  • Click "Saved Templates" tab
  • Click "Import Templates" button
wp-admin/admin.php?page=elementor-templates
04

Import Elementor JSON Files

Upload your downloaded JSON template files through the Elementor import interface.

  • Click "Import Templates" button
  • Select your JSON file from the extracted folder
  • Click "Import Now" to complete the upload
